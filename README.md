## Comandos usados

1. Crear proyecto de React con Vite:

   ```txt
   $: pnpm create vite
   ```

2. Instalación de paquetes:

   ```txt
   $: pnpm i
   ```

3. Instalar paquetes adicionales necesarios:

   ```txt
   $: pnpm i react-router-dom
   ```

4. Levantar proyecto en modo desarrollo:

   ```txt
   $: pnpm run dev
   ```
